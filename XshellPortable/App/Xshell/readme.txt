 1. System requirements
========================
  - Processor		Intel(R) 486DX compatible
  - OS			Windows 95/98/ME or Windows NT/2000/XP
  - Main memory		8M bytes (recommends 16M bytes or above)
  - Hard disk		10M bytes minimum
  - TCP/IP		Winsock 1.1 or later
  - Required software	Internet Explorer 5.0 or later

 2. Key features
===================
  - A powerful 32-bit TELNET/SSH client program.
  - Runs on Windows 95/98/ME and NT/2000/XP.
  - SSH1, SSH2, RLOGIN, TELNET and SFTP protocol connection.
  - Supports serial (RS232C) communication.
  - TCP/IP and X11 forwarding.
  - RSA/DSA public key, Password and Keyboard Interactive authentication methods.
  - Public key generation and management.
  - Manages multiple connections (sessions) using the Session Tab & Session Bar.
  - SSH authentication agent forwarding using Xagent utility 
  - VT100, VT220, VT320, VT382, XTERM, LINUX and ANSI terminal.
  - ZMODEM download and upload.
  - SFTP download and upload.
  - Interoperability with Xmanager(R) and Xftp program.
  - Various connection methods:
    connection dialog, address bar, local command line, shortcut session, and Xactivator.
  - Multi-user configuration.
  - Color scheme edit/import/export feature.
  - Printing selected text, screen, or scroll buffer.
  - International output encoding.
  - Multi-byte support in copy & paste.
  - Local Xshell prompt for advanced users.
  - Innovative User Interface.
  - Session tracing.
  - Keep alive option.
  - Supports the Full Screen mode.


 3. Installation guide
=======================
  You can download Xshell evaluation version at www.netsarang.com.

  Run Xshell30.exe and InstallShield will guide you through the
  installation procedure.
  

 4. How to start Xshell
========================
  You can just double click "Xshell" icon in the Xshell folder.
  For more information about using Xshell, read the "Getting Started"
  section of Xshell Help.
